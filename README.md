# install_scripts
Сборник скриптов для инсталяции программ на все используемые устройства


 Как это запустить?
 
1. Создайте файл скрипта:

nano install_tools.sh

2. Вставьте туда текст скрипта выше. Чтобы сохранить и выйти из nano, нажмите Ctrl+O, затем Enter, а после — Ctrl+X.

3. Сделайте файл исполняемым. По умолчанию созданные файлы не имеют прав на выполнение. 

chmod +x install_tools.sh

4. Запустите скрипт. Так как скрипт изменяет системные пакеты, запускаем его через sudo:

sudo ./install_tools.sh
