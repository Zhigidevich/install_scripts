# install_scripts
Сборник скриптов для инсталяции программ на все используемые устройства


 Как это запустить?
Создайте файл скрипта:

Bash
nano install_tools.sh
Вставьте туда текст скрипта выше. Чтобы сохранить и выйти из nano, нажмите Ctrl+O, затем Enter, а после — Ctrl+X.

Сделайте файл исполняемым:
По умолчанию созданные файлы не имеют прав на выполнение. Исправим это:

Bash
chmod +x install_tools.sh
Запустите скрипт:
Так как скрипт изменяет системные пакеты, запускаем его через sudo:

Bash
sudo ./install_tools.sh
